# AoA-Tool



#### AoA Tool:

**Simple tool for wing AoA variants generation.**



**SETUP:**

To change desired AoA, change values in "newAngles" list.

If you want to see script working, change "swApp.Visible" value to True.



**RUNNING:**

To run scrip you need to have python installed on your computer. 

Open cmd in script directory and type: python "AoA Tool.py"
